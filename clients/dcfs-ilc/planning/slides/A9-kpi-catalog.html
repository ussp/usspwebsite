Appendix: Full KPI Catalog (22 KPIs)

Framework includes 22 KPIs across 5 categories. Up to 8 will be selected based on use cases and data availability.

Note: Some industry KPIs (e.g., code writing speed, lines of code, commit frequency) are excluded — they don't apply to a configuration-first Dynamics 365 program with no autonomous code generation.

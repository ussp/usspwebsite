## Purpose

This guide is the complete reference for Krasan's AI Transformation Framework — a structured approach to augmenting SDLC processes with AI across SAFe delivery teams. It covers every phase from initial governance through ART-wide scale, with practical tools, templates, and measurement methodology.

## Audience

| Audience | Start Here |
|----------|-----------|
| Executive sponsors (CIO, Dinkar) | Ch 1 (Introduction), Ch 2 (Executive Discovery), Ch 9 (Measurement) |

| Team leads / Scrum Masters | Ch 7 (Training), Ch 10 (Playbooks) |
| Individual contributors (BA, Dev, Tester) | Ch 7 (Training — your track), Ch 10 (Playbooks) |
| New engagement setup | Ch 13 (Deployment) |

## Framework Phases

```
Govern → Baseline → Design → Train → Pilot → Measure → Playbook → Scale
```

## How to Use This Guide

1. **Before the engagement:** Read Ch 1-3 to understand the framework, then use Ch 2 (Executive Discovery) to run the initial client conversation
2. **During setup:** Use Ch 4 (Assessment) to gauge readiness, Ch 5 (Baseline) to capture metrics, Ch 6 (Design) to map processes
3. **Before the pilot:** Use Ch 7 (Training) to prepare and deliver role-specific training
4. **During the pilot:** Use Ch 8 (Pilot) for execution cadence, Ch 9 (Measurement) for ongoing tracking
5. **After the pilot:** Use Ch 10 (Playbooks) to codify learnings, Ch 11 (Scale) to expand
6. **For new engagements:** Use Ch 13 (Deployment) to set up everything from scratch

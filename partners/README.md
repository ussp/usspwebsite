# USSP Partner Documents

Store partnership agreements, presentations, reference materials, and tools here.

## Structure

```
partners/
├── krasan/                    ← Krasan Consulting Services (TOPS prime vendor)
│   ├── agreements/            ← Partnership agreements, MOUs, subcontractor docs
│   ├── presentations/         ← Slide decks, pitch materials, PPTs
│   ├── reference-docs/        ← Technical docs, policies, org charts
│   └── shared/                ← Docs shared WITH the partner (external-facing)
│
├── teleray/                   ← TeleRay Inc. (technology / reseller partner)
│   ├── agreements/            ← MOU, partnership terms, NDAs
│   ├── presentations/         ← Pitch decks, capability statements
│   ├── reference-docs/        ← Internal analysis, product info, strategy docs
│   ├── shared/                ← Docs shared WITH TeleRay (external-facing)
│   └── tools/                 ← Interactive tools (valuation calculator, etc.)
```

## Folder Conventions

- **agreements/** — Partnership agreements, MOUs, subcontractor agreements, NDAs
- **presentations/** — PPTs, slide decks, pitch materials, capability statements
- **reference-docs/** — Technical documentation, product info, policies, org charts, internal analysis
- **shared/** — Documents shared externally WITH the partner (treat as external-facing, do not put confidential internal strategy here)
- **tools/** — Interactive HTML tools, calculators, models (open in browser)

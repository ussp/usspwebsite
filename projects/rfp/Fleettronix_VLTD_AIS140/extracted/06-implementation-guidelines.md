# General Implementation Guidelines (Annexure C)

**Source:** Letter VTS Guidelines, Annexure C, Page 17

## 8 Core Guidelines

### 1. Owner Responsibility (Working Condition)

PSV owners must ensure VLT devices are in working condition and regularly send data to the Monitoring Centre via cellular connectivity.

### 2. New Vehicle Registration Block

New PSVs **shall not be registered** in State/UT if not equipped with VLT device + emergency button(s) per CMVR.

### 3. VAHAN Upload for New Vehicles

- Vehicle manufacturer (or authorized agency) uploads Unique ID + details to VAHAN
- Unique ID tagged to vehicle's Engine No. / Chassis No. in VAHAN
- For new PSV registration: VLTD must be **activated and functional**, with Unique ID tagged in VAHAN

### 4. Retro-fitment Process

- VLT device manufacturers / authorized dealers install VLT devices in old PSVs
- Register device + Unique ID in VAHAN **and** corresponding Monitoring Centre systems in real-time

### 5. Enforcement Touchpoints

State/UT enforces fitment + functional status at:

- Registration
- Permit issuance
- Permit renewal
- Fitness check

Applies to VAHAN / respective state vehicle registration / permit system, **and** State/UT Monitoring System.

### 6. Device Activation

- VLT device activated on Monitoring Centre per AIS 140 process
- Functional status updated in VAHAN / state registration software by the MC

### 7. Warranty / Maintenance Requirement

| Vehicle Type | Minimum Support Period |
|---|---|
| New PSVs | 2 years (warranty + cellular connectivity) |
| Other vehicles | 1 year |

### 8. RTO Fitness Check

State/UT ensures CMVR 125H compliance — checks VLT fitment + functional status during fitness certification, in VAHAN and MC.

---

## Note on CMVR 125H

Rule 125H of Central Motor Vehicles Rules, 1989 is the core legal authority for mandating VLTD + emergency buttons in public service vehicles. Enforcement is **State Government responsibility** per MV Act 1988.

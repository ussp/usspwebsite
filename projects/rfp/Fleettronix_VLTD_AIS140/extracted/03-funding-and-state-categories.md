## Scope of Project Cost

Project cost funded under scheme **includes**:

- Backend software (dev + customization)
- Monitoring Centre equipment
- Monitoring Centre space setup
- Training
- Helpdesk support (for vendors + department officials)
- Cloud service (NIC)
- Map services
- State/UT manpower for MC operations
- State-level project management
- National PMU (set up by MoRTH)
- **O&M charges for 2 years post-commissioning**
- State-specific requirements beyond AIS 140 (optional, within project cost)

## Recurring Cost (State/UT bears after year 2)

- Cloud services
- Connectivity & maintenance for MC
- Helpdesk operations
- AMC for application software + hardware
- Annual map services / data

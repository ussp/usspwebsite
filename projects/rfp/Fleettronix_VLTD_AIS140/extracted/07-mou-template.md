# MoU Template — MoRTH ↔ State/UT

**Source:** Letter VTS Guidelines, MoU Section, Pages 20–27
**Title:** *Memorandum of Understanding (MoU) between Ministry of Road Transport & Highways, Govt. of India AND Transport Department, Govt. of {State/UT}*
**Purpose:** Implementation of "Monitoring Centre for Tracking and Monitoring of Public Service Vehicles as per AIS 140 standard under Nirbhaya Framework"

## Parties

- **First Part:** MoRTH (with designation of signing representative)
- **Second Part:** State/UT Transport Department (with designation of signing representative)

## Recitals

- 28 Nov 2016 MoRTH notification mandating VLTD
- Exemption notification 18 Apr 2018
- Cut-off 31 Dec 2018 for old vehicles; 1 Jan 2019 onward mandatory
- Scheme under Nirbhaya Framework — details at Annexure A
- Funding: 100% UT, 90% difficult terrain states, 60% other states

## Clauses

### 1. Effective Date

Date of signing by both parties.

### 2. Objective & Overview

- Improve safety of women + children in PSVs via CMVR 125H
- Equip PSVs with VLT + multiple emergency buttons
- State/UT sets up MC for alert + health monitoring
- Backend processes data + alerts (except emergency button — routed to ERSS)
- MC integrated with ERSS backend for emergency handling

### 3. Scope of Project

- Setting up the Monitoring Centre:
  - Space
  - Backend software development / customization
  - Cloud hosting + deployment on NIC Cloud
  - Map data + services + integration
  - MC equipment
  - Connectivity
  - Helpdesk
- State-level project management (optional PMC)
- Manpower for MC operations
- Training + capacity building
- O&M for 2 years post-commissioning

Only these items count as project cost for funding.

### 4. MoRTH Responsibilities

- Standard guidelines + instructions
- Funding support per Nirbhaya Framework
- Set up National PMU via DIMTS Ltd. (coordination, monitoring, reports, recommend payment release, reconcile UCs)
- VAHAN integration support
- MHA + other dept coordination

### 5. State/UT Responsibilities

- Set up MC per Scheme
- Appoint Implementation Agency
- Timely contribution payment
- Nodal Officer + PIU (single-point contact); optional PMC
- Ensure MC linked to State ERSS
- MC compliant to minimum features per Scheme
- Issue orders/directions to permit holders, VLT mfrs, RTOs for CMVR compliance
- RTO-level enforcement: check fitment + functional status in VAHAN / state registration before registration/fitness
- Integrate MC with VAHAN / state vehicle registration system
- Provide RTO infra (desktops + internet)
- O&M of MC
- Until MC is set up: use any backend system per MoRTH guidelines for CMVR 125H compliance on post-1 Jan 2019 vehicles

### 6. Project Cost & Payment Terms

MoRTH pays X% of actual project cost, capped at ₹Y Cr per Scheme, after deducting proportionate NIC Cloud + National PMU + MoRTH Dashboard costs.

| # | Milestone | Payment % | Timeline |
|---|---|---|---|
| 1 | Signing of MoU | **80%** | T + 15 days |
| 2 | Commissioning of MC by State/UT | — | T + 180 days (by 31 Dec 2020) |
| 3 | 3 months of successful MC operation | **20%** | 3 months post-commissioning |

*T = date of sanction letter issued by MoRTH.*

- Initial 80% = advance
- Subsequent payments = against milestones, verified by MoRTH PMU
- State submits implementation/operation report per milestone
- UC (Utilization Certificate) required per instalment
- MoRTH retains NIC Cloud + National PMU + Dashboard costs proportionately

### 7. Activities After Commissioning

State/UT budgets for full O&M after year 2:

- Cloud services
- Connectivity + maintenance
- Helpdesk operations
- AMC for software + hardware
- Annual map services / data

### 8. Validity

**36 months from Effective Date** — extendable by mutual consent.

### 9. Amendment

Written + signed by both parties only.

### 10. Severability

Invalid parts severed; remainder stays valid.

### 11. Disputes / Arbitration

- First: amicable settlement by signatories
- Failing that: referred to Secretary (Transport), MoRTH — decision is **final**

### 12. Jurisdiction

Laws of India. **Courts in Delhi only.**

### 13. Communication

Written notices in English, registered post or email to the named Representative Authority.

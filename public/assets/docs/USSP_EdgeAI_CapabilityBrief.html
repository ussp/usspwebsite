Platform Overview

VionOS.ai is USSP's Edge AI platform that transforms existing camera infrastructure into an intelligent safety and operations layer. Using NVIDIA Jetson edge devices and advanced computer vision models, VionOS.ai delivers real-time person detection, zone occupancy analytics, and automated alerts — all processed locally on-premises.

The platform is designed for any facility that needs continuous safety monitoring without the privacy risks of facial recognition or cloud-based surveillance. VionOS.ai works with existing IP camera infrastructure and requires no GPU servers or data center equipment.

On-Premises Processing

All AI inference runs on local NVIDIA Jetson Orin devices. Video frames never leave the facility network. No cloud dependency, no external API calls, no data egress.

Privacy by Design

No facial recognition. No biometrics. No PII. Anonymous Person Tokens (SHA-256, 4hr TTL) enable spatial tracking without identification of individuals.

BIPA Compliant

Designed from the ground up to comply with the Illinois Biometric Information Privacy Act. No biometric identifiers or biometric information are collected, stored, or transmitted.

Works with Existing Cameras

Compatible with any IP camera infrastructure via standard RTSP streams. A single Jetson device handles 4-8 camera feeds depending on resolution and analytics complexity.

How It Works

Step 1 — Camera Feeds: Existing IP cameras send RTSP video streams over the facility's local network to NVIDIA Jetson Orin edge devices.

Step 2 — On-Device AI Inference: Computer vision models (YOLO-based person detection, skeletal pose estimation) process video frames in real time directly on the Jetson hardware. No video is recorded, stored, or transmitted externally.

Step 3 — Anonymous Tracking: Each detected person is assigned an Anonymous Person Token — a SHA-256 hashed identifier with a 4-hour TTL. This enables spatial and temporal tracking (zone transitions, dwell time, counting) without any biometric identification.

Step 4 — Alerts & Analytics: The system generates real-time alerts (staff ratio thresholds, restricted zone intrusions, unattended areas) and populates a web-based dashboard accessible on the facility network with zone occupancy data, event logs, and trend analytics.

Privacy Architecture

Zero biometric data. VionOS.ai uses skeletal pose estimation and bounding-box tracking — never facial recognition. Each detected person is assigned an Anonymous Person Token (SHA-256 hash, 4-hour TTL) for spatial tracking without identification. No biometric identifiers or biometric information as defined in 740 ILCS 14/10 are ever collected, stored, or transmitted.

No Facial Recognition

Skeletal pose estimation and bounding-box tracking only. No face detection, no face encoding, no face matching.

No Cloud Transmission

All processing on local NVIDIA Jetson hardware. No external API calls. No data egress from facility network.

Platform Overview

VionOS.ai is USSP's Edge AI platform that transforms existing camera infrastructure into an intelligent safety and operations layer. Using NVIDIA Jetson edge devices and advanced computer vision models, VionOS.ai delivers real-time person detection, zone occupancy analytics, and automated alerts — all processed locally on-premises.

On-Premises Processing

All AI inference runs on local NVIDIA Jetson Orin devices. Video frames never leave the facility network. No cloud dependency.

Privacy by Design

No facial recognition. No biometrics. No PII. Anonymous Person Tokens (SHA-256, 4hr TTL) enable tracking without identification.

BIPA Compliant

Designed from the ground up to comply with the Illinois Biometric Information Privacy Act. No biometric data is collected or stored.

Works with Existing Cameras

Compatible with any IP camera infrastructure via RTSP streams. A single Jetson device handles 4-8 camera feeds.

Privacy Architecture

Zero biometric data. VionOS.ai uses skeletal pose estimation and bounding-box tracking — never facial recognition. Each detected person is assigned an Anonymous Person Token (SHA-256 hash, 4-hour TTL) for spatial tracking without identification.
